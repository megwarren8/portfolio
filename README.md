# Megan Warren, designer and builder

A one-link portfolio: five shipped tools, each shown in its own real colors, with a few of the small decisions that go into making software feel loved.

Open **[index.html](index.html)** in any browser. Fully self-contained: fonts and everything else are embedded, so it renders offline.

## What's linked from here

- [Probability & Matrices](https://probability-matrices-warren.netlify.app), an offline course textbook that reads its own math aloud
- [Grouper](https://grouper-warren.netlify.app), a random student grouper for the classroom
- [Wildcard](https://wildcard-warren.netlify.app), an ice-breaker slot machine
- [Color Companion](https://color-companion-warren.netlify.app), a paint marker color matcher
- [Ask the Turtle](https://ask-the-turtle-brand-kit-warren.netlify.app), a complete brand system built for a nonprofit-fundraising consultancy
